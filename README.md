# How to set up

1. Install NodeJS<br>

    https://nodejs.org/en/download/

    Installed? Check it with:

    ``` bash
    npm -v 
    ```

    1.1. Install Angular

    Run command:

    ```bash
    npm i -g @angular/cli
    ```

    1.2 Install nodepackages

    Run command:

    ```bash
    npm i
    ```

---------------------------------------------

2. Serve webserver

    Run command:

    ```bash
    ng serve -o
    ```
    
    Now a browser window pops up with the website
